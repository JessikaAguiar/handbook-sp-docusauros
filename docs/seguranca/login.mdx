# Login

:::info
Para realizar o login, primeiramente você precisará de um usuário ativo e uma senha, caso não possua, solicite do administrador do sistema.
:::

Para realizar o Login e conseguiar acessar as funcionalidade do sistema, você precisa:

 <ol>
    <li>Inserir o seu usuário no campo <strong>User</strong>;</li>
    <li>Inserir sua senha no campo <strong>Password</strong>;</li>
    <li>Clicar em <strong>Login</strong>;</li>
    <li>O sistema validará seus dados, se estiverem corretos você conseguiará entrar, senão ele emitirá uma mensagem de erro.</li>
 </ol>

 Após realização e validação do login, se este for o seu primeiro acesso, o sistema lhe solicitará a renovação de senha. Neste etapa preencha a uma nova senha seguindo as regras, conforme figura abaixo.
